# Detailing Marketing

SEO-блог для владельцев детейлинг-студий и студий оклейки.

## Команды

```bash
hugo server -D          # Dev-сервер с черновиками
hugo build              # Сборка в public/
./scripts/generate-article.sh "Тема" "ключевики"  # Генерация промпта
```

## Структура

| Путь | Назначение |
|------|------------|
| `content/blog/*.md` | Статьи |
| `layouts/` | Hugo-шаблоны |
| `prompts/` | Промпты для генерации |
| `scripts/` | Скрипты автоматизации |
| `static/` | Статика (изображения) |

## Генерация статей

1. Запустить `./scripts/generate-article.sh "Тема" "ключевики"`
2. Скопировать промпт в Claude
3. Сохранить результат в `content/blog/`

Гайд по стилю: `prompts/article-guide.md`

## Стиль контента

- Разговорный тон, без воды
- Конкретика: цифры, примеры, пошаговые инструкции
- Развенчивание мифов
- CTA в конце — ссылка на Telegram
