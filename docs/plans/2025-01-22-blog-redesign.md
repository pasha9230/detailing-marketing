# Редизайн блога Detailing Marketing

## Цель

Превратить минималистичный блог в полноценный с обложками статей и рубриками.

## Решения

### Визуальный стиль
- **Направление:** современный минимализм
- **Ширина контента:** 600px
- **Шрифт:** системный sans-serif (Inter, -apple-system) вместо monospace
- **Цвета:**
  - Фон: `#ffffff`
  - Текст: `#1a1a1a`
  - Акцент (ссылки, рубрики): `#0066cc`
  - Мета-информация: `#666666`

### Обложки статей
- Генерируются через Gemini на основе содержания статьи
- Хранятся в `static/images/`
- Указываются в front matter: `cover: "/images/название.jpg"`
- Соотношение сторон: 16:9

### Рубрики
Две рубрики вместо 9 кластеров:

| Рубрика | Включает темы |
|---------|---------------|
| **Привлечение клиентов** | Маркетинг, SEO/карты, Соцсети, Платная реклама, Сайт |
| **Развитие детейлинг центра** | Бизнес, Ценообразование, Позиционирование |

## Структура сайта

```
/                             → редирект на /blog/
/blog/                        → список всех статей с фильтром
/blog/привлечение-клиентов/   → статьи рубрики
/blog/развитие-студии/        → статьи рубрики
/blog/название-статьи/        → отдельная статья
```

## Компоненты

### Главная страница (`/blog/`)
- Шапка: "Detailing Marketing" + автор + Telegram
- Фильтр по рубрикам (кнопки: Все / Привлечение клиентов / Развитие студии)
- Карточки статей в одну колонку

### Карточка статьи
```
┌─────────────────────────────┐
│ [обложка 16:9]              │
├─────────────────────────────┤
│ ПРИВЛЕЧЕНИЕ КЛИЕНТОВ        │  ← рубрика мелким
│ Заголовок статьи            │  ← крупный, жирный
│ 21 января 2025              │  ← дата серым
│ Первые две строки описания  │
│ статьи для превью...        │
└─────────────────────────────┘
```

### Страница статьи
- Хлебные крошки: Блог → Рубрика
- Обложка на всю ширину (600px)
- Заголовок + мета (дата, автор, рубрика)
- Контент статьи
- CTA в конце (ссылка на Telegram)

## Техническая реализация

### Front matter статьи
```yaml
---
title: "Заголовок статьи"
date: 2025-01-21
description: "Описание для превью и SEO"
cover: "/images/article-cover.jpg"
category: "привлечение-клиентов"
tags: ["яндекс карты", "локальное seo"]
---
```

### Изменения в hugo.toml
```toml
[taxonomies]
  category = "categories"
  tag = "tags"
```

### Файлы для изменения
- `hugo.toml` — добавить taxonomies
- `layouts/_default/baseof.html` — новые стили
- `layouts/blog/list.html` — карточки с фильтром
- `layouts/blog/single.html` — обложка + рубрика
- `layouts/partials/card.html` — компонент карточки (новый)
- `content/blog/yandex-karty-dlya-detailinga.md` — добавить cover и category

## Воркфлоу создания статьи

1. Написать статью в `content/blog/`
2. Скинуть текст в Gemini: "Сгенерируй минималистичную обложку для статьи про..."
3. Сохранить картинку в `static/images/`
4. Указать `cover` и `category` в front matter
5. `hugo server -D` для проверки
6. Коммит и деплой
